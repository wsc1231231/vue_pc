# 第一百二十五章 他在论坛上发了一个帖子

回剑阁的第三天傍晚，叶尘下线后回到自己的出租屋。

他没立刻开电脑。他坐在沙发上，打开了那本他这一年里一直在更新的本子。本子已经写得很厚，最新的一页停在「彼界将受冲击」六个字下面。

他用一支水笔在那六个字下面又加了一行：「我是第一个倒下的人。」

写完之后他在原地坐了很久。

他想，他这一边的世界，他这一年里其实没怎么管过。他没和母亲说过他和「九州武道录」的关系。他没和同事说过。他甚至没和自己说过。他每天上线下线，把「九州武道录」当成自己的另一份工作，做完就睡。

但今天他突然意识到一件事。他这一边的世界，也不是「绝对安全」的。如果顾苍穹的计划成了，这一边的「精神」会被吸。具体怎么吸，吸完什么样，叶尘不知道。他只知道，他这一年里习以为常的「下班回家煮泡面」「夜里看手机」「周末看电影」的日子，有可能突然消失。

他必须做一件事，把这件事告诉别人。

但是告诉谁？

告诉他母亲？母亲不信这种「另一界」的事。母亲会觉得他生病了。

告诉同事？同事听完会笑十年。

告诉医生？医生会让他吃药。

他叹了一口气，打开电脑。

他在浏览器里搜「另一界 连接 现实」。出来的全是网络小说。他又搜「跨界 通道 真实」。出来的还是小说。他又搜「玩家 操控 NPC 真实」。这次出来的是一些科幻论坛的讨论。

他点开一个叫「彼岸轨」的小众科幻论坛。这个论坛他大学的时候用过，注册账号还在。论坛里大部分用户都是科幻爱好者，平时讨论平行世界、量子物理、未解之谜，氛围比微博、贴吧理性一些。

叶尘在论坛上发了一个新帖。

标题：「假设你发现你玩的手游连接的是一个真实世界，你会怎么做？」

正文他写得很简单：

「不是 SAO 那种灵魂封进游戏。是反过来，你在手机这一头按按键，按键的指令通过某种东西传到另一边，让另一边一个真实的人按你的指令做出动作。这个人不是『AI 模拟』，是真的。她有自己的意识，有自己的情绪，有自己的故事。」

「她的世界要被她那边的某个反派毁掉。这个反派的计划如果成功，你这一边的世界也会被影响。」

「你会怎么做？」

帖子发出去之后，叶尘把电脑关了，吃晚饭，洗澡。

晚上九点他重新打开电脑。帖子下面有了十一条回复。

第一条：「楼主 SAO 看多了。」
第二条：「这剧本写得不错，可以投稿《科幻世界》。」
第三条：「兄弟你是不是该去医院看一下脑子，我有个心理咨询师朋友可以推荐。」
第四条：「真的话？真的话我先把那个反派一刀剁了。」
第五条：「请问怎么和另一边的人交流？文字对话框还是语音？」
第六条：「你不是来骗稿费的吧。」

叶尘把每一条都读了一遍。大部分是玩笑，少数是认真的「讨论设定」。他正要关掉，第十条回复出现了。

那一条回复的 ID 是「观测者」。

「观测者」这个 ID 头像是黑的，注册时间显示「七年」。

回复的内容只有一句：

「如果是真的，你应该想想为什么是你被选中。」

叶尘盯着这一句看了很久。

他打开「观测者」的主页。主页空空的，没有任何发帖记录，只有这一条回复。注册七年，发帖零，今天突然冒出来回了他这一句。

他立刻给「观测者」发了一条私信。

「你是谁？」

发出去之后，他盯着屏幕，等了二十分钟。没回。

他又发了一条：

「你怎么知道我可能是真的？」

还是没回。

他又发了一条：

「请告诉我一些事。任何事。」

仍然没回。

叶尘把电脑屏幕调小，靠在椅背上，呼出一口气。

他想，「观测者」这种 ID，注册七年没发帖，今天突然冒出来回他这一句话。这种人在论坛上不是「随便路过」。这种人是在「等」。等一个特定的帖子。等一个特定的提问。今天他这个帖子，触发了对方等的那个条件。

但是对方又只回了一句话。回完就消失。

叶尘把手指压在桌面上，慢慢敲。

他想到了一件事。

他打开自己一年前用过的「九州武道录」的初始截图。那张截图是他第一次启动游戏时的弹窗。弹窗上写着一行字：

「您已被选为本产品唯一测试者。」

「唯一」两个字。

「测试者」三个字。

他这一年里一直把这一行字当成「开发组的营销噱头」。今天「观测者」那一句话突然让他想，「唯一」不是营销词。是实情。整个九州武道录，全世界只有他一个人在「玩」。

那「测试者」是什么？是「试」这件事的人？「试」什么事？

他在本子上又写了一行字：

「我是唯一被选中的人。被谁选？为什么是我？」

写完他把笔放下。

那一夜他在沙发上躺了很久，盯着天花板。

他这一年里很少这样躺着思考。他平时下班回家，先吃饭，再上线，再睡觉，日子按节拍走。今天他被「观测者」那一句话戳到了一个他平时不愿意想的问题，

「为什么是我？」

他这种人，普通的程序员，平凡的家庭，三十年来没做过任何「大事」。他甚至没有谈过一段超过半年的恋爱。他每天上下班，地铁车厢里上百号人，他是最不起眼的那一个。

但是天命剑心选中了他。

为什么？

他没有答案。

天蒙蒙亮的时候他爬起来，给「观测者」又发了一条私信：

「我会一直问。等你哪天想回，我就在。」

发完之后他关掉电脑，倒头睡了三个小时。

中午十二点他醒来，第一件事是打开手机看「九州武道录」。沈鹿溪在剑阁的练剑场上练剑，她出剑的角度比一周前更稳，剑光带的火焰附魔也更熟。叶尘看着她，心里那个「为什么是我」的问题，慢慢从「问题」变成了「答案的雏形」。

也许，是因为他愿意。


那一周里叶尘每隔一晚就去论坛看一眼。

「观测者」没再回他。但是奇怪的事情发生了。

第三天叶尘的帖子下面，多了一条新的回复。回复人的 ID 叫「拙工」。

「拙工」的回复也很短：

「楼主，你描述的这种『手机连接异界』的现象，二十年前有人在另一个论坛提过。那个论坛已经关了，那个人也消失了。你不是第一个。」

叶尘的心跳了一下。

他立刻私信「拙工」。「拙工」这一次回了：

「我只能告诉你，二十年前那个人也是『唯一测试者』。他用的不是手机，是早期的 PDA。他描述的『另一界』和你描述的不一样，但是结构很像。」

「他后来去哪了？」叶尘问。

「拙工」过了很久才回：「不知道。某一天他不再上线，账号也注销了。」

叶尘盯着屏幕，呼吸有一下没有顺。

二十年前那个「唯一测试者」，最后注销账号。这意味着两种可能。一种是「他放弃了」，回到普通生活。第二种是，他「过去」了。也就是，他用某种方法跨到了那一边，留在了那一边，再没回来。

叶尘合上电脑，在心里把这一条新信息写到墙上的证据清单上。

「我不是第一个被选中的人。」他写，「但我是这一代的唯一一个。」

那天晚上他打开「九州武道录」，沈鹿溪在剑阁的瀑布边坐着。她朝屏幕外说：

「叶尘，你今天的心跳，比昨天稳了一些。」

「嗯。」叶尘说，「我今天找到了一点线索。」

「什么线索？」

叶尘想了想，把「拙工」的事告诉她。沈鹿溪听完，沉默了一下，然后她问：

「叶尘，那个二十年前的『唯一测试者』，是不是和我妈妈认识的『天命』，是同一个人？」

叶尘愣了一下。

「鹿溪，你能不能问问你妈妈，她三十年前知不知道有一个『另一界』的人，曾经和她或者她那一辈的人有过类似的连接？」

「嗯。我明天问。」
