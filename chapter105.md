# 第一百零五章 碎片的共鸣

战斗结束了。

骨老的尸体被火烧成了灰——舍命丹的爆炸把他烧得连骨头都没剩下。山寨大半被烧毁——剩下的天枢阁守卫早就趁乱跑了。被解救的实验体全部被周明渊护到了山林深处。有些走得动——有些走不动——但他们都活着。

战斗结算面板：

【任务完成：探查天枢阁分坛（任务被自发任务覆盖）】
【自发任务完成：解救实验体】
【解救人数：87人（含37人神识破碎不可救者）】
【经验：+25000】
【声望：+2000（剑阁）+5000（江湖白道）】
【特殊评价：以一己之力救活50人 - 触发隐藏成就】
【掉落物品：见尸体搜刮】

但叶尘没在意奖励。他在意的是——分坛深处还有一件事。

他和沈鹿溪一起，慢慢走进了山寨的中央楼。这是骨老平时驻扎的地方。一楼是审讯室——审讯室的墙上还留着血迹，地上散乱着几具实验体的尸体（应该是审讯失败后被骨老直接处死的）。二楼是他的住处——一间陈设极简的房间，只有一张床、一个书桌、一个灵石柜。书桌上有一本日志——是骨老每天记录实验进度的。叶尘没翻——他知道这本日志里全是血腥的数据。但他在屏幕外让沈鹿溪把日志收进背包——也许以后林晚照查天枢阁内部档案时能用到。

地下有一层。叶尘的小地图上标着一个金色的感叹号。

"那里。"叶尘说。

地下密室。

打开门的瞬间——叶尘和沈鹿溪同时屏住了呼吸。

密室的中央——有一座青铜柱。柱子上插着一颗发光的碎片。

【物品检测：天命剑心碎片（七分之一·完整片）】

是另一片天命剑心碎片。比沈鹿溪体内的那一片更完整、更大、光芒更亮。

这是天枢阁正在试图"激活"的另一片碎片。他们用这片碎片做"信号源"——去探测江湖上其他碎片的位置。同时他们也在用这片碎片做"实验源"——用它来验证人体能不能"激活"碎片。

叶尘看着那片碎片——呼吸停了。

他想——他能不能拿走它？拿走之后会不会和沈鹿溪体内的碎片合并？

意识深处的沈鹿溪走过去——叶尘没操控她——她是自己走过去的。

她伸手——触碰了那颗碎片。

碎片在她触碰的瞬间——发出了一道剧烈的金光。

【物品反应：碎片识别"同源"】
【碎片自动飞入：沈鹿溪体内】
【天命剑心碎片融合中……】

沈鹿溪整个人被金光包围。她的身体在剧烈颤抖——叶尘能感觉到她的胸口在剧烈翻腾。两片碎片在她体内融合——这是一种比"被夜袭"更剧烈的物理反应。

【角色情绪：剧痛（极强）】

叶尘的瞳孔瞪大。他想给她用"还魂丹"——但还魂丹已经用完了。他没有能立刻让她"不疼"的东西。

他在屏幕外说："鹿溪，撑住，我在。"

她在金光中微微点头。她没出声——她疼得说不出话——但她在咬牙撑。

融合持续了三分钟。

这三分钟对叶尘来说像三个小时。他看着屏幕上沈鹿溪的金光颜色一会儿变深一会儿变浅——每一次变深都是融合在"加剧"。他不能做任何事——他没有道具能阻止融合，他也不能强行让沈鹿溪从这种融合中退出（强退会反噬，可能直接死）。他只能看着她疼。

他在屏幕外把手按在胸口——通过双心跳——他也能"感受"到一部分她的疼。这种"感受到的疼"不是肉体的疼——是灵魂层面的撕裂感。和她的肉体疼相比当然轻得多——但叶尘还是觉得难受。他想替她疼——但他做不到。意识共鸣只能"传递信号"，不能"转移痛苦"。这是叶尘第一次恨自己的"无力"。

三分钟后——金光散去。沈鹿溪倒在地上——她浑身是汗——但她睁着眼睛。她活着。

【天命剑心碎片·融合完成】
【新觉醒度：50% → 70%】
【新增被动：意志共鸣（中级）】
【新增被动：天命剑意（低级）】
【角色基础属性·全面提升】

叶尘看着面板上的属性：

【角色属性变化】
- 根骨：3 → 6（提升一倍）
- 经脉损坏度：85% → 30%（修复了55%）
- 真气海容量：760 → 1200
- 攻击：198 → 280
- 防御：124 → 180
- 速度：175 → 240

"……"叶尘看着那个"经脉损坏度"——从85%降到了30%。

经脉损坏度——是沈鹿溪八岁那年被天枢阁封筋脉的"后遗症"。这九年来她一直是"废物"——根本原因就是这个数字一直是85%。今天——融合一颗碎片——这个数字直接降到了30%。

她不是废物了。

她从今天起是一个真正的真气后期天才剑修。等再融合几片碎片——她可能会成为江湖上前所未见的强者。

她比她妈妈年轻时还要强。

意识深处的沈鹿溪也感受到了。她从地上爬起来，活动了一下身体。她的每一根经脉都比之前通畅。她甚至能感觉到——她体内的真气海洋比之前广阔了一倍多。

她举起手——掌心朝上。一缕金色的真气在她掌心凝成了一个小光球。这是她以前做不到的——以前她的真气连"具象化"都做不到，全靠秋水剑作为"载体"。今天她可以让真气直接在掌心成形——这是真气巅峰修者才能做到的事。

她看着自己的手。这双手不再是"废物的手"。这双手是——天命剑心的双持者的手。

她突然笑了——不是那种眼眶湿的笑——是一种发自内心的、简单纯粹的开心。她朝着屏幕外的方向——做了一个手势——她比划了"二"，然后做了一个"还有五片"的口型。

叶尘读懂了。她在说："还有五片。"

意思是：再找五片，她就能彻底解封。

意思是：再找五片——她就能强到天枢阁阁主也奈何不了她。

意思是：再找五片——她和叶尘就能一起去找她妈妈。

叶尘也笑了。他在屏幕外用语音说："好。我们一起找。"

她点头。

那天晚上叶尘下线之前——沈鹿溪坐在山林里的一棵树下休息。周明渊在旁边照顾被解救的实验体——他在给一个老人喂水、给一个孩子包扎手腕。这些事不是他平时会做的——他是剑阁外门首席——他的"训练"全部是"打架"。但今天他在做这些"细活"——一个不到二十岁的剑修——居然有耐心给一个孩子包扎得很仔细。叶尘在屏幕外看着——觉得这个NPC在他心里的形象又升了一截。

叶尘看着沈鹿溪——心情很复杂。

她经历了今天的战斗：救人、面对大宗师巅峰的BOSS、用舍命丹自爆BOSS、融合碎片——任何一件事都足以让一个人崩溃。但她坐在树下——眼神平静——甚至有点开心。

她在变。

不是变得"更强"——是变得"更稳"。她在战斗里学会了"不慌"。她在救人里学会了"不退"。她在面对蚀魂术的时候学会了"为他付出"。她已经不是那个一年前在剑阁后山被一只灵兽追到哭的小姑娘了。

她长大了。

叶尘看着她，心里有一种难以言喻的感受——既骄傲（她长大了），又心疼（她长大的代价是什么？是无数次的死亡、是无数次的差点失去）。

他在屏幕外用语音说："鹿溪，你今天打得很好。"

她朝着屏幕外笑了一下。她比了一个手势——把右手按在胸口，然后伸出来，朝向屏幕。

她在说"我把心交给你"。

叶尘也把右手按在胸口——然后伸出来——朝向屏幕。

【信任度：95 → 96】

那一晚周明渊把被解救的人安顿好之后——沈鹿溪和他一起踏上了回剑阁的路。她的身体比之前任何一天都更轻盈、更有力量。

她披着那件浅米黄色的斗篷——走在山道上。月光从云层后面照下来——把她的影子拉得很长。她的影子不再是一个"废物的影子"——是一个"剑修的影子"。

叶尘在屏幕外——第一次对她萌生了一种说不清的、但是非常温柔的情绪。

他在心里说："傻丫头，长大慢一点。"

意识深处的沈鹿溪好像听到了。她朝着屏幕外又笑了一下。

【信任度：96 → 97】
