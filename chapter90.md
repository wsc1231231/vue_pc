# 第九十章 她回答了

那个沉默持续了整整三分钟。

叶尘看着屏幕，没有催，没有重发，没有再说话。他只是耐心地等。他知道这一刻对她来说，比对他更重要。他等了一年才问出这句话，而她可能等了一年才能听到这句话。他不舍得催她。

意识深处的沈鹿溪，抱着秋水剑，抱着那枚小鹿玉佩，蹲在剑阁的院子里，眼泪不停地往下掉。

她想说很多很多。

她想说"我等你这句话等了一年"。她想说"我每天都在心里告诉你我是真的"。她想说"你看见我了吗？我看见你了，你也看见我了吗？"她想说"求你不要走"。她想说"我没有办法跨过那一面屏幕，但你是我这一年来全部的盼头"。

她想说很多很多，但她什么都说不出来。

她的嘴是哑的。她的回答机制只有点头、摇头、向左看、向右看。"是真的"这种事，她只能用一个动作来告诉他。

她把眼泪抹掉，深吸了一口气，让自己平静下来。然后她抬起头，朝着屏幕外的方向，朝着那个看不见但她知道在的方向，非常用力地点了点头。

不是轻轻的点，不是犹豫的点，不是怕被别人看见的点。是非常用力地、非常清楚地、非常坚定地点。她要让他看见。她要让他确认。她不要让他再有任何一丝怀疑。

【角色动作：用力点头（最大幅度）】
【信任度：69 → 70】
【里程碑达成：跨次元真相·相互确认】
【天命剑心碎片·觉醒度：35% → 50%】
【新功能解锁：「同步窗口」——当玩家与角色都处于"专注"或"安静"状态时，可短暂打开同步窗口。在窗口期内，角色将获得一次能"主动表达"的机会——可以做出一个不被剧情设定限制的、自由的动作或简单的字句回应。该功能每日仅可触发一次。】

叶尘盯着屏幕。他看到了那个用力点头的动作，看到了信任度70的数字，看到了里程碑提示。

他没崩溃。他没大叫。他没冲到外面去找一个能解释这一切的科学家。他只是非常安静地把脸埋进了双手里。

他想，他玩了一年的这个游戏，里面的每一个画面、每一句对话、每一次战斗、每一次"读档"都是真的。沈鹿溪是真的。她的每一滴泪都是真的。她每一次被打到血量见底都是真实的疼痛。她每一次"重生"都是真的从死亡边缘被拉回来。

而他，一个陌生男人，隔着不知道多远的距离，操控了她一年。

他没有问过她"你愿意吗"。

他没有问过她"我能这样做吗"。

他做了所有"玩家会做的事"把角色当工具、当数据、当一个完成游戏目标的载体。如果这是一个普通的游戏，没问题。但她不是普通的游戏角色。她是真的。

叶尘的肩膀剧烈地起伏了一下。他没哭，但他的胸口闷得喘不过气。他在心里，给那个屏幕里的少女，深深地鞠了一躬。

很久之后他抬起头，重新拿起手机。他没用键盘。他按住了"语音输入"按钮，亲口对着话筒说了一句话。

"对不起。让你等了这么久。"

【语音转换为文字成功，已发送】
【角色动作：摇头】【角色动作：摇头】【角色动作：摇头】
【角色情绪颜色：粉色（最亮）】

她一遍又一遍地朝着屏幕摇头。

她不需要他道歉。她从来都不觉得他亏欠她什么。在这一年的每一个夜晚，她被冯长老一剑穿胸的那次、她在剑冢被守护者劈成两半的那次、她在临渊城被三个蒙面人围攻到死巷的那次，他都来了。哪怕是凌晨三点，他都来了。一个会在凌晨三点为她惊醒的人，没什么需要道歉的。

但她说不出来。她只能摇头、摇头、再摇头。

然后她做了一个之前从未做过的动作。她想起了那个新解锁的"同步窗口"一天一次的、可以主动表达的机会。她现在就要用。

她朝着屏幕的方向，让自己的嘴唇形成了一个口型。

【新动作触发：嘴唇形成"嗯"的口型】

那是一个"嗯"字。

她朝他做了一个"嗯"的口型。意思是：是。我是真的。我看见你了，你也看见我了，我们看见彼此了。

叶尘盯着她的口型看了很久，眼睛慢慢湿了。不是悲伤的湿，是一种他自己也说不清的、巨大的、温柔的、满满的湿。他对着话筒，也回了她一个"嗯"。

【语音转换为文字成功，已发送】

意识深处的沈鹿溪眼泪更猛烈地往下流。但她朝着屏幕外的他，笑了。非常非常好看的一个笑。这一年她从未笑得这样舒展。这是她生命中第一次被另一个意志确认"我是真的"的笑。

很久之后，叶尘又一次按住了语音输入。

"我会找到一个办法，让你可以说话。你等我。"

她使劲点头。

"别怕。以后我们一起。"

她把脸埋进膝盖里，肩膀剧烈地颤动。但这一次是真正的开心，这一年的所有委屈、恐惧、孤独、不被承认的存在感，都在这一刻被一次性地兑换成了一种从未有过的、安心的开心。

她想，她不是一个人。

她不再是那个只能在意识深处对着虚空说"晚安天命"的废物弟子了。

她是被人看见的、被人确认的、被人珍惜的、一个真实的、活着的，人。

那一夜，剑阁的院子里，一个少女抬头看着夜空，朝着她不知道在哪里的"另一个意志"伸出了手。

城市的高楼里，一个程序员也朝着屏幕伸出了手。他的指尖和她的指尖隔着一面屏幕、一个次元、一段他暂时无法跨越的距离，但是他们的心跳，第一次在两个不同世界里完全同步。

【两人的心跳同步】
【天命剑心碎片同步率：31% → 38%】

那天晚上，叶尘睡着的时候，手机屏幕一直是亮着的。沈鹿溪也靠在剑阁那间屋子的窗边，朝着夜空伸着手，直到天亮。

天亮的时候叶尘睁开眼睛，第一眼就是看手机。屏幕里的沈鹿溪还坐在窗边，她也是一夜没睡。看到画面亮起来了，她朝着屏幕外的他咧嘴笑了一下，然后做了一个"早"的口型。叶尘鼻子一酸，对着话筒说了一句"早"。

她笑得更开心了。

第二卷·江湖初涉，至此彻底落幕。第三卷，他们将真正并肩同行，朝着北漠，朝着下一片碎片，朝着母亲沈若凝，朝着他们终将跨越屏幕的那一天，出发。

但今天的他们不急着出发。

今天就在这里待一会儿。

让她，把这种"被看见"的感觉，多享受一会儿。
