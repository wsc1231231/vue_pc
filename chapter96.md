# 第九十六章 她看到了

第二天清晨，叶尘上线得比平时早。他打开手机的时候，沈鹿溪还在睡觉，披着那件浅米黄色的斗篷，脸埋在被子里，睡颜很安静。

叶尘没操控她。他只是看了一会儿。然后他注意到一件事：她的枕头底下露出了一角折好的粗纸。

叶尘出于好奇心，操控沈鹿溪轻轻翻了个身。她迷迷糊糊地动了一下，但没醒。然后那张纸被翻了出来。叶尘小心地把它展开。

是一幅画。

画得很糙：一个小人，眉毛画得太粗，眼睛画得太大，下巴上有几个点。歪歪扭扭，比例失调，一看就是不会画画的人画的。

但叶尘看着那张画，心里某个地方"咯噔"了一下。

"……这是我？"

他一开始不敢确定。但他看着那个小人下巴上的几个点，是胡渣。他下意识地摸了摸自己的下巴。今天早上他没刮胡子。

她见过他。她把他画下来了。

叶尘的手抖了一下。他赶紧把那张纸折好，按原样塞回枕头底下。然后他下意识地操控沈鹿溪也躺回床上，好像什么都没发生过一样。

但他自己坐在屏幕外，盯着那张已经被收回去的画的位置看了很久。

他想，她什么时候见过他？怎么见的？她记得他的样子？

他翻了翻昨天的对话日志，想找线索。然后他注意到了昨天的那条系统提示：

【天命剑心碎片·觉醒度：35% → 50%】
【新增被动：意志共鸣（低级）】

意识共鸣。他想起来：他升级碎片的那一刻，沈鹿溪在意识深处感觉到了什么。但他当时没多想。现在他突然意识到，也许那次升级让她"看到"了他。

叶尘把手机贴在胸口。胸口那颗"双心跳"轻轻跳着，一快一慢，节奏分明。他能感觉到她的心跳——她还在睡，所以心跳很慢、很稳。

很久之后，他用语音输入说了一句话："早。"

沈鹿溪在床上动了一下。她睁开眼睛——叶尘看到她的瞳孔从模糊变得清晰，然后她的目光朝着屏幕外的方向看了一眼。她笑了。

她朝他做了一个"早"的口型。

叶尘心里那种奇怪的感觉更加强烈了：她确实见过他。她现在看着他的眼神不再是那种"朝着虚空"的眼神，而是"朝着具体的某个方向"。她知道他大概在哪个角度。

她坐起来，懒懒地伸了个懒腰。然后她从枕头底下摸出那张画。

叶尘呼吸停了一下。

她把那张画展开，看了一会儿，然后从床头摸出毛笔。她蘸了点水（她没有墨，剑阁不会给一个废物弟子准备墨），在画上又添了几笔。

她在改下巴上胡渣的位置。

叶尘看着她的动作，突然意识到：她在凭记忆"修正"那张画。而她每次修正，画上的那张脸都更像他一点。

他看着她画到一半，停下来，皱眉，又画了一笔，再停下来，再皱眉。她在和那个画里的他较劲，好像那个人的鼻梁应该再高一点，眼睛应该再窄一点，下巴的形状应该是椭圆而不是方形。

她画了一个时辰。

最后那张画比第一版更接近叶尘了。但还是有些奇怪，她记忆里的他大概不是这样。她叹了口气，把笔放下，把画重新折好，塞回枕头底下。

然后她拿起秋水剑，准备出门练剑。

叶尘在屏幕外，眼眶湿了。

那天叶尘没有让沈鹿溪"看到"他知道她画了画。他装作什么都没发生过。但他在自己的备忘录里加了一段话：

"她见过我。她在画我。她记不清我的样子，所以她画了又改，改了又画。她想认识我。她想看见我。"

"我也想见她。"

"想去她那个世界，亲自走到她面前，告诉她：'画里这个人，就是我'。"

他保存了备忘录，关上手机。

他坐在阳台上看了很久的天空。今天的天空是淡蓝色的，有几朵很薄的云。他想，她那个世界的天空今天也是这种颜色吗？她的剑阁今天有云吗？她出门练剑的时候，会不会被风吹到？

他第一次在现实里为屏幕里的那个人担心了一下天气。

那天晚上叶尘下线之前，他做了一件他从未做过的事：他打开手机的相册，找到了他那张身份证的扫描件。他截图了，把照片发给了游戏对话框。

【系统提示：图片格式不支持。】

他试了几次。每次都失败。

他叹了口气。他知道这游戏不让他发图片。但他还是想把自己的脸给她看一眼。她画了一整天还画不像，他想让她不必再画。

他在对话框打了一句话："你画的人，画得越来越好了。"

意识深处的沈鹿溪突然瞪大眼睛。她不知道叶尘是怎么知道的，但他知道了。她吓了一跳，下意识地把脸埋进斗篷里。

过了一会儿，她探出头来，朝着屏幕外的方向，做了一个"对不起"的口型。

叶尘看到那个口型，笑了。他打了一句话："不用对不起。我很喜欢。"

她朝他笑了一下，眼眶又红了。

【信任度：77 → 78】

那天晚上她睡觉的时候，把那张画放在了枕头边，而不是枕头底下。她不再藏了。

意识深处，沈鹿溪闭着眼睛，但没睡着。

她在心里反复念叨："天命，天命，我画了你。你叫什么名字？你今年多大？你那个世界长什么样？"

这些问题她问不出来。但她想，总有一天，她要亲口问他。

她笑了一下，闭着眼睛笑的，不是给任何人看的笑。

她想——她不再害怕"天命"这个名字了。这个名字以前对她来说是一种"超越她理解的力量"。但现在不是了。现在他对她来说，是一个有具体五官的、坐在发光板子前的、会因为她紧张就紧张的、普通的人。

一个普通的人。

她把胸口那颗碎片按了一下。两个心跳轻轻同步着。她想，这样就好。这样她就不孤独了。

她睡着了。

【信任度：78（睡眠期间稳定）】
