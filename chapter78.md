# 第七十八章 新技能树

第二天早上，叶尘睡到自然醒——周末的特权。他懒洋洋地翻了个身，摸到床头的手机。

打开游戏的瞬间——

【欢迎回来！】
【您离开期间发生的事件】角色已休息一夜；角色情绪：平静；体力恢复：100%。

"……'角色情绪'这个东西，自从信任度50之后就一直显示着。"叶尘琢磨，"以前我都没注意——现在不能不注意了。"

他退出主界面进入游戏。沈鹿溪此时正在内门弟子的食堂吃早饭——一碗粥、两个馒头、一碟咸菜。叶尘看着这个画面皱了皱眉。

"师父也太抠了，武会冠军了，还吃这种早饭？"

他想了想，操控沈鹿溪走出食堂。"先去训练场。武会奖励的天品丹药——是时候吃了。"

凌霜剑阁的内门高阶训练场是一座建在山腰上的青铜大殿，门口有两座持剑石像，殿内是一片宽阔的场地。地面铺着特殊的青玉石板，可以承受真气巅峰强度的攻击。

叶尘操控沈鹿溪走进训练场。里面只有两三个内门弟子在练剑——见到沈鹿溪进来，他们立刻停下了动作。

"师姐。"

"鹿溪师姐。"

叶尘看着头顶的称呼变化乐了一下。"以前进训练场没人理我，现在'师姐''师姐'地叫。这就是冠军的待遇。"

他没理那些弟子，操控沈鹿溪走到训练场最里面——那里有一间独立的"丹房"，是供弟子服食丹药、稳定境界用的。

进入丹房，叶尘点开背包：

【物品：天品·凝元丹】
【描述：天品丹药。服用后可直接突破一个小境界。】
【效果】真气中期 → 真气后期

"突破！"叶尘选择了"使用"。

屏幕上的沈鹿溪盘膝坐在丹房中央的蒲团上，把那枚晶莹剔透的丹药放进了嘴里。突破开始——她身上开始浮现出柔和的金色光晕。光晕从胸口扩散到全身，像是潮水一样向四周涌动。空气里出现了若有若无的剑鸣声，是真气在体内运转的副产物。

【境界：真气中期·圆满 → 真气后期·初阶】

伴随着突破——

【角色等级：18 → 22】

直接提升了四级！

【属性提升】
- 攻击：156 → 198
- 防御：87 → 124
- 速度：143 → 175
- 真气：520 → 760

"四十多点的属性提升！"叶尘喊出声，"这一颗丹药，比我之前打几十只野怪获得的属性还多！"他看着这一片提升的数字乐得合不拢嘴。但他没意识到——突破过程中，意识深处的沈鹿溪也在感受这股真气。

每一次境界的突破，对她来说都是一次身体的重塑。她的经脉变得更加宽阔、更加通畅，丹田里的真气海洋也变得更广阔。天命剑心碎片在她的丹田深处微微颤动——它似乎也在配合这次突破。

她在心里轻声说——

"天命，我又强了一点。"

"下次我们再战斗的时候，你会感觉到的。"

她不知道叶尘能不能听到。但叶尘——他确实感觉到了。突破完成的那一瞬间，他的"情绪感应"功能传来了一缕极其鲜明的情感。

【角色情绪：喜悦】

不是简单的"开心"，是那种成长之后的、带着小小骄傲的、想要分享的喜悦。

叶尘盯着那个情绪标签。"……她这是在跟我'报告'吗？"他没想到的是——确实是。

"喂。"叶尘鬼使神差地在对话框里打了一句话，"恭喜你突破了。"

沈鹿溪在意识深处整个人亮了。她听到了。"天命"主动跟她说话了。不是那种"打怪""刷副本""boss太硬"的玩家用语，是——"恭喜你突破了。"像一句普通的、人和人之间的关心话。

她的眼眶热了一下。她在心里轻声回答——

"嗯。谢谢你。"

虽然她知道叶尘听不到，但她还是说了。

【信任度：55 → 56】

突破完之后叶尘很满足，开始研究新装备。秋水剑已经很好用了。但他注意到武会还奖励了一些"剑诀"——可以理解为剑法心法。他打开背包翻看：

【剑诀：流水十二式】（已学）
【剑诀：基础刺击】（已学）
【剑诀：基础格挡】（已学）

"这几个都是低阶剑法。我现在真气后期了——"

叶尘点开了剑诀面板的最上方。那里有一个新解锁的位置：

【剑诀：凌霄九式·待学习】
【学习条件：师父亲授】

"凌霄九式——昨天看到的那个剑阁顶级心法！"叶尘的眼睛亮了，"得找师父，'凌霄九式'必须由师父亲授。"

他立刻操控沈鹿溪走出训练场，朝着韩青松的木屋走去。走到半路，韩青松的身影出现在了前方的小径上，仿佛在等她。

"师父。"沈鹿溪停下脚步。

韩青松抬头看了一眼她："你突破了？"

"……是的。"

"很好。"老人点头，"那么，是时候教你下一步的东西了。跟我来。"

韩青松转身往后山走去。叶尘操控沈鹿溪跟上。

【触发主线：凌霄九式·传授】

"啊哈——"叶尘乐了，"系统居然预判我要去找师父？这NPC剧情太智能了。"

他操控沈鹿溪跟着韩青松一路往后山深处走。走过一片竹林，过了一座小桥，绕过一座石壁——眼前豁然开朗。一道瀑布从悬崖上倾泻而下，激起漫天水雾。瀑布下方是一汪深不见底的潭水。潭水边上有一座古色古香的小阁楼。

【新地点：凌霄阁】

"凌霄阁，昨天看到的'武会冠军解锁'区域！"叶尘惊叹，"这地方的特效——"晨雾、瀑布、潭水、古阁，每一个细节都做得无可挑剔。

韩青松走到瀑布前，回过头："鹿溪。"

"嗯？"

"从今天起，我教你凌霄九式。这是凌霜剑阁的镇阁剑法。三百年来，只有十二个人完全学会过——其中包括我。"

老人停了一下，又补了一句："你母亲也学会过。"

老人转身指向瀑布边的石壁。那石壁上密密麻麻刻着名字。

【剑魂榜】

叶尘把镜头拉近。榜上排在最前面的那一行：

【沈若凝——天人境·秋水流光剑·三百年来第二人】

叶尘看着那行字，沉默了几秒。然后他转头看向沈鹿溪此刻站立的位置——她的侧脸映在朝阳里，像一柄还未出鞘的剑。

"你母亲的名字，刻在那里。"韩青松的声音很轻，"我希望，将来有一天，你的名字也能刻上去。"

沈鹿溪深吸了一口气："我会努力。"

韩青松笑了笑："那我们开始吧。"

老人拔出了腰间那柄旧剑。那柄剑是沈若凝二十年前送给他的。剑身上的铭文，叶尘第一次看清：

"青松不老，鹿溪不歇。"

那是当年沈若凝亲手刻上去的。背面还有一行小字："为我老友，永远的朋友。"

叶尘看着那行字，又看了看老人现在的样子——白发苍苍，背微微弯，但握剑的手稳如磐石。二十年。一个朋友离开二十年没有回来。一个朋友守着她留下的孩子十七年。一柄剑挂在腰间二十年没有换过。

叶尘把屏幕亮度调低了一些——他怕自己又眼睛湿。

"开始吧。"屏幕里的沈鹿溪也举起了秋水剑。
