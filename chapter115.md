# 第一百一十五章 筋脉解封（下）

天亮的时候，解封完成了。

最后一道封印破开的瞬间，沈鹿溪的胸口爆发出一道金色的光柱。光柱直冲屋顶，把客房的天花板都映成了金色。沈若凝在女儿对面已经精疲力尽，她瘦弱的脸上全是汗。但她最后用一道纯净的真气，把女儿的"清醒锚"切断，同时把女儿身体里残余的封印能量全部清出。

【筋脉解封·完成】
【经脉损坏度：10% → 0%】
【根骨：6 → 8（再次提升）】
【真气海：扩容100%】
【境界：大宗师巅峰 → 暂时无境界（待重塑）】

【特殊事件：境界重塑】
【描述：长时间的封印解除后，角色境界需要重新结算。重塑结束后，新境界将基于角色真实潜力。】

沈鹿溪睁开眼睛。她的眼睛比之前任何时候都要明亮。她活动了一下身体，每一条经脉都"通透"得像水管被洗过。她以前从来不知道自己的身体可以"这么轻"。

她下意识地拿起秋水剑，走到了客房外面的庭院里。

朝阳刚刚升起。剑阁的庭院里有一棵已经百年的老梅树。沈鹿溪站在梅树下，举起秋水剑。她想试一下。

她出了一剑：凌霄九式·第一式·松迎客。

剑势从下而上斜削。剑尖在抬起的过程中划出三个微小的弧线。每一道弧线都精准地命中"判定点"。而且，剑光带着一丝肉眼可见的金色剑意。

"……"沈鹿溪盯着剑尖。

剑光不应该是金色的。她以前练这一招，剑光是青色的（秋水剑的颜色）。今天的金色，是天命剑心碎片的剑意。

她又出了一剑：凌霄九式·第二式·鹤回首。

剑势反转。剑光在空中划出半圆，这一次的金色剑意更浓。半圆的最后一点甚至切开了梅树旁边一片飘落的梅花花瓣，从中间一分为二。

沈鹿溪愣住了。

她以前出第二式的时候，剑光只能在空中"成形"，没有任何实际的"切割能力"。今天，她随手出的第二式居然能切开飞行中的花瓣。

这种力量她从未有过。

她又试了第三式·松鹤鸣。

剑势"S"形。剑光在空中划出的"S"切开了梅树上挂着的三片叶子。三片叶子同时飘落，切口整齐得像被人用尺子量过。

她没多想，直接换了"双剑模式"。秋水剑加青松剑。

【凌霄九式·合势第七式·松柏长青】

两柄剑同时旋转。剑光形成一个直径一尺的金色圆环。圆环朝着梅树的主干飞过去——

"咔嚓"。

梅树主干上多了一道整齐的环形切口。深度约一寸，并未完全切断，但是整棵树被这一招"晃"了一下。如果她再用一分力，这棵百年梅树就被她拦腰斩断了。

沈鹿溪收剑站在原地。

她以前从来没体验过这种"力量"。这种力量不是叶尘操控时给她的，她现在没有被操控。叶尘下线了，他陪了她一夜，已经累得睡着了。

这一切的力量是她自己的。

她举起手，掌心朝上。一缕真气在她掌心凝成一道淡淡的金色光球。这是"真气具象化"，只有大宗师巅峰以上的修者能做到。她以前做不到。今天她做到了。

她做到了。

她在庭院里站了很久。她抬头看着天空。天很蓝，朝阳从云层后面照过来。

她想，她从今天起不再是那个废物。也不再是那个"靠天命操控才能赢"的剑修。她是一个真正的、靠自己的力量站在凌霄武道顶点的、剑修。

她的眼眶又湿了。但这一次的湿不是悲伤、不是感动，是一种"我做到了"的纯粹的喜悦。

中午，叶尘上线。

他先看了一眼任务面板：【筋脉解封·完成】。然后他看了一眼角色面板：

【角色属性变化】
- 根骨：6 → 8
- 经脉损坏度：10% → 0%
- 真气海容量：1200 → 2400
- 攻击：280 → 450
- 防御：180 → 320
- 速度：240 → 380
- 境界：大宗师巅峰（重塑后稳定）

"……"叶尘看着那一片暴涨的属性。

"……根骨8？！真气海翻倍？！"

他差点把手机扔出去。

"什么情况？！我角色一觉醒来就变成了超级赛亚人？！"

他立刻横过手机，准备试一下手感。

【训练场·进入】

他随手按了一个"出剑"。意识共鸣触发。他和沈鹿溪共同发力。

剑光从沈鹿溪的手中射出，在训练场的空气中划出了一道整整十丈长的金色剑光。剑光打在训练场墙壁上的木靶上，木靶被一剑斩成两半。

"……卧槽。"叶尘盯着屏幕。

"我以前练这一招打不出这种威力。"

但是他立刻反应过来：是因为根骨翻倍、真气海翻倍。这一切的属性提升让他的"操作"加"她的力量"产生了完全不同的效果。

他兴奋地试了几个连招。每一招都比之前强得多。沈鹿溪的身体在他的操控下变成了一个真正的"超级赛亚人"模式。

但是叶尘试到第五个连招的时候，他停下来了。

他注意到了一件事：沈鹿溪的"自主修正"已经几乎不再发生了。

以前他的"按键操作"会有偏差，是沈鹿溪用她自己的"剑意感知"做微小修正，让动作落在"完美判定点"上。但是今天，他的"按键操作"和角色的"动作"是完全一致的。没有"修正"，没有"误差"。

"……？"

叶尘想了一下，他明白了。

不是沈鹿溪不修正了。是沈鹿溪已经能"提前判断"他的指令并主动配合了。她不需要"修正"，因为她从一开始就和他的指令"完全同步"。

【意志共鸣·进入完全融合阶段】

意识深处的沈鹿溪在被操控的同时闭着眼睛。

她在感受。

她在感受叶尘的操控节奏：他这个人的操作有他独特的"心跳"。她已经能"听到"他的"心跳"，他的下一个指令她可以提前1秒预判。

她在感受叶尘的情绪：他今天很开心，因为她变强了。她能从"心跳"里听出他的开心。

她在感受叶尘的存在。

她不再像以前那样只是"被操控的容器"。她现在是"和他一起的搭档"。

她很安心。

叶尘在屏幕外说了一句话："鹿溪，你现在的力量比我想象的还强。"

她朝着屏幕外笑了一下。

意识深处她说："我们现在是真正的一对剑了。"

【信任度：100（满值，状态：已稳定）】
【新里程碑：自我·觉醒】
【天命剑心碎片·觉醒度：90% → 95%】
